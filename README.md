# SungML

> A machine learning library built completely from scratch.

SungML is a long-term open-source project focused on:

- Implementing classical machine learning algorithms from scratch
- Understanding every algorithm at the implementation level
- Benchmarking against scikit-learn
- Researching and developing novel machine learning architectures

---

## Project Goals

- Learn machine learning beyond APIs
- Build a production-quality ML library
- Explore original machine learning algorithms
- Publish research ideas and experiments
